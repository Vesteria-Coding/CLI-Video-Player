## IMPORTANT!
In order for sound to work you need ffmpeg installed and added to path.

# Installation & Usage:
Run these commands to install and run CLI-Video-Player:
```bash
git clone https://github.com/Vesteroa-Coding/CLI-Video-Player.git
cd CLI-Video-Player
pip install -r requirements.txt
python main.py /Path/To/MP4/
Player:
```

### CLI-Video-Player
This is tested on both windows and linux.
